# mysecond2
Gams 처럼 탭 제목이나 아이콘 바꿀수 있는거 만듬
 # 오프라인 게임 모음 런처

## 사용법
1. ZIP 다운로드해서 풀기
2. index.html 더블클릭 → 메뉴 화면 뜸
3. 새 게임 추가 방법:
   - games 폴더 열기
   - 새 .html 게임 파일 넣기 (파일 이름 = 게임 제목)
   - index.html 새로고침(F5) → 자동으로 새 버튼 생김!
4. 탭 제목/아이콘 바꾸고 싶으면:
   - index.html 파일 열기 (메모장이나 텍스트 앱으로)
   - 맨 위 <title>과 <link rel="icon"> 부분 수정 → 저장
5. 게임 제거: games 폴더에서 파일 지우기 → 새로고침
